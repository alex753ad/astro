Шаг 2 — Создай файл скрипта
Вставь это целиком в PowerShell:

powershell

$content = @'
$source = "C:\Users\User\Desktop\ASTRO"
$backupRoot = "C:\Users\User\Desktop\clone astro"
$maxCopies = 24

$timestamp = Get-Date -Format "yyyy-MM-dd_HH-mm"
$dest = Join-Path $backupRoot $timestamp

New-Item -ItemType Directory -Force -Path $dest | Out-Null
robocopy $source $dest /MIR /R:3 /W:5 /LOG+:"$backupRoot\backup.log"

$copies = Get-ChildItem $backupRoot -Directory | Sort-Object Name
if ($copies.Count -gt $maxCopies) {
    $copies | Select-Object -First ($copies.Count - $maxCopies) | Remove-Item -Recurse -Force
}
'@
$content | Out-File -FilePath "C:\Scripts\backup.ps1" -Encoding UTF8

Задача уже запустится сама — но нужно донастроить
Текущая настройка запускает задачу каждый час от момента создания. После перезагрузки ПК расписание может сбиться. Вот как это исправить:

Пересоздай задачу с правильными флагами (от администратора)
powershell

# Удалить старую задачу
Unregister-ScheduledTask -TaskName "HourlyBackup_ASTRO" -Confirm:$false

# Создать новую — правильную
$action = New-ScheduledTaskAction `
  -Execute "powershell.exe" `
  -Argument "-NonInteractive -WindowStyle Hidden -File C:\Scripts\backup.ps1"

$trigger = New-ScheduledTaskTrigger `
  -RepetitionInterval (New-TimeSpan -Hours 1) `
  -Once -At "00:00"

$settings = New-ScheduledTaskSettingsSet `
  -StartWhenAvailable `
  -ExecutionTimeLimit (New-TimeSpan -Minutes 30) `
  -RestartCount 3 `
  -RestartInterval (New-TimeSpan -Minutes 1)

Register-ScheduledTask `
  -TaskName "HourlyBackup_ASTRO" `
  -Action $action `
  -Trigger $trigger `
  -Settings $settings `
  -RunLevel Highest `
  -Force
Что делает каждый флаг:

Флаг	Что делает
-StartWhenAvailable	✅ Если пропустил запуск пока ПК был выключен — запустится сразу при включении
-ExecutionTimeLimit	Ограничение на 30 мин (на случай зависания)
-RestartCount 3	Перезапустит задачу 3 раза если вдруг упала с ошибкой
Сценарии:

Ситуация	Что произойдёт
ПК включён постоянно	Копия каждый час ✅
Выключил в 14:00, включил в 17:00	Копия сделается сразу при включении ✅
Перезагрузка	Задача продолжает работать автоматически ✅
Проверить что задача активна:
powershell

Get-ScheduledTask -TaskName "HourlyBackup_ASTRO" | Select-Object TaskName, State
Должно показать State: Ready — всё работает 👍
