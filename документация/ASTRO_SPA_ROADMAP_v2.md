### 🎯 Конверсионная воронка: Free → Pro

> Ключевой принцип: пик желания платить возникает **во время** взаимодействия с продуктом, а не до него.

#### Онбординг без барьеров

- ⬜ **Анонимное построение карты** — до регистрации:
  - Пользователь вводит дату/время/место рождения → карта строится → отображается
  - Только после этого: "Сохраните карту — войдите или зарегистрируйтесь"
  - Анонимная сессия хранится в localStorage 24 часа
  - Это убирает главный killer конверсии — регистрацию до получения ценности
- ⬜ **Google OAuth** — снизить friction до 1 клика:
  - `POST /auth/google` → JWT токен
  - Сохранить анонимную карту под новым аккаунтом автоматически
- ⬜ **Интерактивный онбординг первой карты:**
  - Tooltip на ASC: "Асцендент — ваша маска для мира, то как вас воспринимают с первого взгляда"
  - Tooltip на MC: "Середина Неба — ваше призвание и публичный образ"
  - Tooltip на аспектах: "Красные линии — напряжённые аспекты, синие — гармоничные"
  - Dismiss после первого просмотра, сохранять в localStorage

#### Точка конверсии 1 — конец краткой интерпретации

- ⬜ **Смысловой обрыв** — не технический блок, а нарративный обрыв:
  - Интерпретация заканчивается на эмоционально значимой теме (7-й дом, Луна, Венера, Нептун)
  - Пример текста обрыва: *"Особое влияние на вашу жизнь оказывает Сатурн в 7-м доме. Этот аспект формирует специфический паттерн в близких отношениях, который объясняет..."*
  - Текст обрывается на многоточии, дальше — blur-эффект на следующих 3 строках
- ⬜ `PaywallModal.jsx` — компонент с тремя вариантами появления:
  - После краткой интерпретации
  - При клике на заблюренные транзиты
  - При попытке сохранить второй профиль на Free
- ⬜ **Содержимое PaywallModal:**
  - Заголовок: "Продолжите разбор с полной интерпретацией"
  - 3 конкретные выгоды с иконками:
    - 🔮 "Полный разбор личности, карьеры и отношений — 2000 слов вместо 500"
    - 📅 "Ваши личные транзиты на 6 месяцев вперёд"
    - ⭐ "Лучшие даты для важных решений каждую неделю"
  - CTA кнопка: "Попробовать Pro бесплатно 7 дней"
  - Подпись: "Затем $12/мес или $99/год · Отмена в любой момент"
  - Ссылка "Уже есть аккаунт? Войти"

#### Точка конверсии 2 — preview транзитов

- ⬜ **Реальный граф транзитов с частичным blur:**
  - Показывать настоящий timeline, рассчитанный для пользователя

### 📧 Email-маркетинг — онбординг и конверсия

> Email — самый недооценённый канал в астро-нише. Персональные транзит-письма открывают 40–55%.

#### Инфраструктура

- ⬜ Подключить **Resend** (рекомендуется) или **Postmark** для транзакционных писем:
  - Resend: от $0 за 3000 писем/мес, простой API, React Email шаблоны
  - Настроить SPF, DKIM, DMARC для домена
- ⬜ Создать систему шаблонов на **React Email** или **MJML:**
  - Универсальный базовый шаблон с хедером (логотип + астро-символика)
  - Welcome, Retention Day 2, Retention Day 7, Trial Ending, Weekly Digest, Transit Alert

#### Цепочка онбординга (Free-пользователь)

- ⬜ **Welcome-письмо (сразу после регистрации):**
  - Тема: "Ваша натальная карта готова, [Имя]"
  - Содержание: краткий инсайт из карты пользователя (Солнце в знаке + 1 предложение), ссылка на карту
  - CTA: "Открыть карту"
  - Не продавать в первом письме — только дать ценность

- ⬜ **Retention-письмо День 2 (если не перешёл в Pro):**
  - Тема: "[Планета] влияет на вашу [сферу] прямо сейчас"
  - Содержание: реальный актуальный транзит из карты пользователя
  - Пример: "Сегодня Венера входит в ваш натальный Юпитер. Это один из лучших периодов года для [сфера применения]. Полный прогноз на следующие 3 месяца — в Pro."
  - CTA: "Смотреть полный прогноз" → PaywallModal
  - Техническая реализация: Celery task в момент регистрации с delay 48 часов; берёт ближайший транзит из Transit Engine

- ⬜ **Retention-письмо День 7 (если всё ещё Free):**
  - Тема: "Вы пропустили N важных транзитов за эту неделю"
  - Содержание: список из 3 реальных транзитов, которые были активны, с датами и кратким значением
  - Фрейм потери (loss aversion работает лучше, чем обещание выгоды)
  - CTA: "Не пропускать следующие транзиты" → 7-дневный trial

- ⬜ **Письмо День 14 (последняя попытка):**
  - Тема: "Специальное предложение для вас — 30% скидка на первый месяц"
  - Stripe Coupon на 30% скидку с истечением через 48 часов
  - Один промокод на одного пользователя (проверка через Stripe Customer ID)

#### Цепочка для Pro-пользователей (retention)

- ⬜ **Еженедельный персональный дайджест (каждый понедельник):**
  - Тема: "Ваша астрологическая неделя: [дата] – [дата]"
  - Содержание:
    - 3 лучших дня недели для [работа / отношения / финансы] с конкретными датами
    - Активные транзиты этой недели: название + краткое значение + дата пика
    - Лунные фазы недели
    - Одна рекомендация Planner Engine ("избегайте крупных решений в среду — Меркурий квадрат вашему Сатурну")
  - Это не напоминание об app — это **продукт, который приходит сам**
  - Открываемость таких писем в астро-нише: 50–70%

- ⬜ **Транзит-алерт (триггерное письмо):**
  - Когда Юпитер, Сатурн или Уран начинают проходить натальную планету пользователя — уходит письмо
  - Тема: "Важный период начинается в вашей карте"
  - Для Юпитера/Венеры — позитивный фрейм ("один из лучших периодов года")
  - Для Сатурна/Марса — подготовительный фрейм ("период требует осознанности")

- ⬜ **Письмо за 3 дня до окончания триала:**
  - Тема: "Ваш пробный период заканчивается через 3 дня"
  - Показать что именно потеряет пользователь: сколько транзитов будет скрыто, сколько слов в интерпретации

### 💑 Синастрия — главный growth-driver Фазы 2

> Синастрия — самая виральная функция в астрологии. "Посмотри нашу совместимость" приводит второго пользователя.

- ⬜ **Бэкенд: `POST /api/v1/chart/synastry`:**
  - Принимает два chart_id (или два набора birth data)
  - Рассчитывает межаспектную сетку: аспекты между планетами партнёров
  - Возвращает: список аспектов, силу связи по 5 категориям, общий "score совместимости"
  - Категории: Коммуникация (Меркурий), Страсть (Марс/Венера), Эмоциональная связь (Луна), Долгосрочность (Сатурн), Рост (Юпитер)
- ⬜ **AI-интерпретация синастрии** (отдельный промпт):
  - Не "вы подходите друг другу" — а "эта связь особенно сильна в сфере X, напряжение возникает в Y"
  - Потоковый стриминг, те же секции что у натальной карты
- ⬜ **UI синастрии:**
  - Две карты рядом с линиями межаспектов (цвет = тип аспекта)
  - Radar chart (D3.js) с 5 категориями совместимости
  - Секции интерпретации с возможностью "поделиться" каждой секцией
- ⬜ **Виральный sharing синастрии:**
  - Shareable карточка совместимости: "Наш астрологический портрет" — PNG 1200×630px
  - Публичная ссылка на синастрию (оба участника видят без логина, но не могут скачать PDF)
  - Invite-механика: "Отправьте ссылку партнёру — пусть добавит свои данные рождения"
- ⬜ **PDF-отчёт совместимости** (Premium):
  - 10–15 страниц: карты обоих, межаспектная таблица, AI-интерпретация по категориям
  - Брендированный дизайн: пара планет на обложке, фирменный стиль

- ⬜ **A/B тест промптов:**
  - Два варианта промпта → случайное распределение пользователей → сравнение рейтингов
  - Промпт с лучшим рейтингом становится основным через 2 недели
- ⬜ **Раздельные промпты по уровню пользователя:**
  - `novice`: простой язык, аналогии, избегать терминов
  - `enthusiast`: средний уровень, можно использовать астрологические термины
  - `expert`: профессиональный язык, все термины, орбы, минутные дуги

#### Секционный стриминг

- ⬜ **Структурированный стриминг по секциям:**
  - Бэкенд отдаёт SSE-события с типом секции: `{"type": "section_start", "section": "sun"}`
  - Фронтенд рендерит heading перед стримингом каждой секции
  - Порядок: Солнце → Луна → Асцендент → Меркурий → Венера → Марс → Юпитер/Сатурн → Транзиты
  - Пользователь видит прогресс, не ждёт целиком — снижает воспринимаемое время ожидания

#### Оптимизация расходов

- ⬜ **Кеширование по секциям** (не полного текста):
  - Ключ кеша: `{chart_id}:{planet}:{sign}:{house}:{prompt_version}`
  - Если Солнце в Овне в 1-м доме уже интерпретировалось — берём из кеша
  - Оценочная экономия: 40–60% токенов для популярных сочетаний
- ⬜ **Умный выбор модели:**
  - Free-пользователи: всегда DeepSeek (дешевле в 10 раз)
  - Pro: GPT-4o для новых интерпретаций, DeepSeek для повторных запросов той же секции
  - Premium: всегда GPT-4o

#### Новые AI-функции

- ⬜ **Ежемесячный AI-прогноз:**
  - 1-го числа каждого месяца Celery task генерирует прогноз для всех Pro/Premium пользователей
  - Тема: главные транзиты месяца + персональные рекомендации по неделям
  - Доступен в app + приходит в email

- ⬜ **"Спроси астролога" — AI-чат в контексте карты:**
  - RAG на `knowledge_base.json` + натальная карта пользователя как system context
  - Пользователь спрашивает: "Почему у меня проблемы с деньгами?" → AI анализирует 2-й и 8-й дом
  - Лимит: 10 вопросов/мес для Pro, безлимит для Premium
  - Это killer-feature для удержания Premium — пользователь чувствует персонального астролога 24/7

- ⬜ **Вторичные прогрессии (Secondary Progressions):**
