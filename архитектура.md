# Astrea Timeline — Архитектура проекта (v3.0)

Документ описывает текущую архитектуру системы. Источник истины — код в репозитории.

## 1. Назначение

Astrea Timeline — веб-приложение для построения натальных карт, расчёта транзитов, лунного календаря и генерации AI-интерпретаций с использованием Swiss Ephemeris.

**Стек**: Python 3.12 + FastAPI (backend), React 18 + Vite (frontend), PostgreSQL + Redis.

Монетизация — подписочная модель **4 тарифов: Free / Lite (790₽) / Pro (1990₽) / Premium (7990₽)** через Stripe. Trial отсутствует.

| Свойство | Реализация |
|---|---|
| Точность расчётов | Swiss Ephemeris (pyswisseph), < 1 угл. сек |
| Стриминг текста | Server-Sent Events (секционный, с `section_start`/`section_end`) |
| Устойчивость к сбоям LLM | GPT-4o → DeepSeek → шаблоны |
| Контроль расходов | Daily budget + per-tier rate limits |
| Безопасность | JWT + Google OAuth + bcrypt |
| Планировщик | house_passages (бисекция) + AI |
| Лунный календарь | Фазы, знаки, рекомендации |
| Email | Resend API + SPF/DKIM/DMARC |

## 3. Технологический стек

### Backend
- **Python 3.12** (Docker: python:3.12-slim)
- **FastAPI** — асинхронный веб-фреймворк
- **SQLAlchemy 2.0** + **Alembic** — ORM и миграции БД
- **PostgreSQL 16** (psycopg2-binary) — основная БД
- **Redis** — кэширование (transit_cache, geo_cache, interp_cache)
- **pyswisseph** — Swiss Ephemeris для астрономических расчётов
- **Pydantic 2.7** + pydantic-settings — валидация данных
- **python-jose** — JWT-токены
- **passlib[bcrypt]** — хеширование паролей
- **slowapi** — rate limiting
- **httpx** — асинхронный HTTP-клиент
- **pytz** + **timezonefinder** — работа с часовыми поясами
- **geopy** — геокодирование координат
- **Celery** — фоновые задачи (retention emails, PDF-генерация)

### Frontend
- **React 18.3** + **React Router 6.23** — SPA фреймворк
- **Vite 5** — сборщик модулей
- **Tailwind CSS 3.4** — утилити-фреймворк стилей
- **D3.js 7.9** — интерактивное SVG-колесо зодиака

### Инфраструктура
- **Docker Compose** — локальная разработка (PostgreSQL + Redis + API)
- **Railway** — production backend + Celery worker + Cron jobs
- **Vercel** — production frontend
- **GitHub Actions** — CI/CD пайплайны
- **Resend** — транзакционные письма (домен: astreatime.ru)

## 7. API Endpoints

### Chart Management
- `POST /api/v1/chart/calculate` — расчёт натальной карты
- `GET /api/v1/chart/{id}` — получение данных карты
- `POST /api/v1/chart/save-anonymous` — сохранение анонимной карты после логина
- `DELETE /api/v1/profile/charts/{id}` — удаление карты

### Interpretation
- `POST /api/v1/chart/{id}/interpret` — AI-интерпретация (SSE-стриминг)
- `GET /api/v1/profile/history` — история интерпретаций

### Transits & Forecast
- `GET /api/v1/chart/{id}/transits` — транзиты за период (требует lite+)
- `GET /api/v1/chart/{id}/transits/interpret` — AI-обзор транзитов (SSE, требует pro+)
- `POST /api/v1/chart/{id}/transits/event/interpret` — интерпретация одного транзита
- `GET /api/v1/chart/{id}/transits/async` — асинхронный расчёт (Celery)
- `GET /api/v1/chart/{id}/forecast/daily|weekly|monthly` — прогнозы

### Planning
- `GET /api/v1/chart/{id}/planner/monthly` — планировщик на месяц (pro+)

### Calendar
- `GET /api/v1/calendar/lunar` — лунный календарь с фазами

### Authentication
- `POST /api/v1/auth/register` — регистрация по email/пароль
- `POST /api/v1/auth/login` — вход в аккаунт
- `POST /api/v1/auth/refresh` — обновление JWT токена
- `POST /api/v1/auth/google` — Google OAuth вход
- `GET /api/v1/auth/me` — текущий пользователь
- `DELETE /api/v1/auth/me` — полное удаление аккаунта (GDPR)

### Payments
- `POST /api/v1/payments/checkout` — создание Stripe Checkout сессии
- `POST /api/v1/payments/portal` — Stripe Customer Portal
- `POST /api/v1/payments/webhook` — Stripe webhook обработчик
- `GET /api/v1/profile/subscription` — информация о подписке (Feature Flags)

### Internal (Railway Cron, X-Internal-Secret header)
- `POST /api/v1/internal/onboarding-emails` — отправка Day2 и Day7 писем
- `POST /api/v1/internal/weekly-digest` — еженедельный дайджест (пн 09:00 МСК)
- `POST /api/v1/internal/coupon/generate` — генерация Day14 купонов 30%

### Health & Monitoring
- `GET /health` — базовая проверка
- `GET /health/db` — проверка БД
- `GET /health/ai` — проверка LLM сервиса

## 8. Email система

### Сервис: Resend
- **Домен**: `astreatime.ru` (SPF + DKIM + DMARC)
- **FROM**: `noreply@astreatime.ru`

### Шаблоны (`backend/email_service.py`)

| Функция | Триггер | Аудитория | Содержание |
|---|---|---|---|
| `send_welcome_email` | Первая натальная карта | Все | Солнце в [Знак] + инсайт |
| `send_retention_day2` | Celery, 48 часов спустя | Авторизованные | Ближайший активный транзит |
| `send_retention_day7` | Celery, 7 дней спустя | Free только | Loss-aversion: N пропущенных транзитов |
| `send_retention_day14` | Celery, 14 дней спустя | Free только | Купон 30% на годовой Lite (24ч, один раз) |
| `send_payment_failed_email` | Webhook invoice.payment_failed | Pro+ подписчики | Ссылка на Stripe Portal, grace 3д |
| `send_weekly_digest` | Railway Cron пн 09:00 МСК | Pro/Premium | 3 дня недели + активные транзиты + луна |
| `send_transit_alert_email` | Медленная планета начинает | Pro+ | Юпитер/Сатурн/Уран/Нептун проходит |

### Celery retention chain
При первой натальной карте авторизованного пользователя:
1. **Day 2** (48ч): реальный транзит из Transit Engine
2. **Day 7** (7 дней): список 3 пропущенных транзитов (только free)
3. **Day 14** (14 дней): Stripe Coupon 30%, один на пользователя, хранится в `coupons_sent`

## 9. Конверсионная воронка

### Фаза 1: Онбординг (День 1)
- Тултипы при первом просмотре карты (ASC → MC → Аспекты)
- Флаг `astrea_onboarding_seen` в localStorage
- Привлечение к натальным домам и аспектам

### Фаза 2: Нарративный обрыв (День 1-2)
- AI-интерпретация обрывается на эмоционально значимой теме
- Выбор темы: 7-й дом → отношения, Луна → эмоции, иначе → Сатурн
- Blur + CTA "Читать полную интерпретацию" → **free_to_lite**

### Фаза 3: Paywall Modal (День 2)
При `403 { error: "tier_required", required: "..." }`:

| Контекст | Заголовок | CTA | Цена |
|---|---|---|---|
| `free_to_lite` | "Прочитайте полную интерпретацию" | Lite | 790₽/мес |
| `lite_to_pro` | "Разблокируйте ваши транзиты" | Pro | 1 990₽/мес |
| `pro_to_premium` | "Работаете с клиентами?" | Premium | 7 990₽/мес |

### Фаза 4: Retention (День 2-14)
- Day 2 email: активный транзит
- Day 7 email: важные события (только free)
- Day 14 email: купон 30% (только free)
- Weekly digest: еженедельные рекомендации (pro+)

## 13. Безопасность

| Компонент | Реализация |
|---|---|
| Секреты | Только в .env (не в коде) |
| Пароли | bcrypt с солью |
| API CORS | Whitelist доменов через CORS_ORIGINS |
| JWT | access 15 минут, refresh 7 дней, ротация через JWT_SECRET_PREV |
| Stripe webhook | Проверка цифровой подписи |
| SQL injection | SQLAlchemy ORM параметризованные запросы |
| Rate limiting | slowapi + per-tier ограничения |
| Anti-fraud Premium | SlowAPI 3 req/min + IP-мониторинг Redis (сброс на 3+ IP за 30м) |
| Circuit breaker | AI_DAILY_BUDGET_USD лимит → fallback на шаблоны |
| GDPR | Полное удаление аккаунта с CASCADE |
| Internal endpoints | X-Internal-Secret header проверка |

## 17. Типичный user flow

1. **Анонимный пользователь**
   - Попадает на лендинг (HomePage)
   - Вводит данные рождения
   - Видит натальную карту (без сохранения)
   - Paywall: "Создайте аккаунт" → AuthModal

2. **Free пользователь**
   - Регистрируется / входит
   - Карта сохраняется в БД
   - Видит AI-интерпретацию (обрывается на теме)
   - Paywall: "Upgrade to Lite" → Stripe checkout

3. **Lite подписчик**
   - Может создавать 5 карт в месяц
   - Видит полные интерпретации (800 слов)
   - Может просматривать транзиты (список)
   - Лунный календарь на 12 месяцев

4. **Pro подписчик**
   - Неограниченно карт
   - AI-интерпретации 2500 слов, 15/месяц
   - Видит интерактивные транзиты + AI-обзор
   - Планировщик на месяц
   - PDF-отчёты (5/месяц)

5. **Premium подписчик**
   - Полная функциональность
   - RAG-чат на базе натальной карты
   - CRM клиентов
   - PDF-отчёты 50/месяц
